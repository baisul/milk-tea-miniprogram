# milk-tea-miniprogram
# 项目描述：该项目是由云函数和云数据库开发的一款奶茶点单小程序。
# 项目功能介绍
# 用户可选择某个店铺选择商品后下单（目前尚未做支付功能）。
# 功能模块：
# 1.收货地址管理：用户可管理自己的收获地址。
# 2.饮品分类管理：管理员可管理饮品的分类与饮品。
# 3.热门饮品维护：管理员可管理热门饮品。
# 4.店铺维护：管理员可管理店铺。
# 5.订单管理：用户可管理自己下的订单。
# TODO：管理员功能尚未做，目前所有用户都可以看到所有的功能菜单。

# 本地启动项目
# 一、申请一个小程序
# 1.先在微信公众平台注册https://mp.weixin.qq.com，申请一个小程序，找到appId。
# 2.申请完成后，再用微信开发者工具打开milkTea项目，要填写刚申请到的appId才能打开项目。

## 二、初始化云开发环境
# 1. 在微信开发者工具中打开本项目
# 2. 点击「云开发」按钮，开通云开发并创建环境。
# 3. 在 `app.js` 中将 `env: 'your-env-id'` 替换为你的云环境ID。
# 4. 在project.config.json中将appId替换成你的appId，cloudbaseCollectionName和          envId替换成你的云环境ID。

## 三、创建数据库集合
在云开发控制台 > 数据库中，创建以下9个集合：

| 集合名 | 说明 |
|--------|------|
| shops | 店铺 |
| categories | 饮品分类 |
| drinks | 饮品 |
| orders | 订单 |
| addresses | 收货地址 |
| favorites | 店铺收藏 |
| hotDrinks | 热门饮品 |
| users | 用户信息 |
| shopping_cart | 购物车信息 |

## 四、设置数据库权限

在云开发控制台 > 数据库 > 每个集合 > 权限设置：

- ** 选择「自定义安全规则」，设置为：
  {
    "read": true,
    "write": true  
  }
## 五、部署云函数

# 1. 在微信开发者工具中，右键点击 `cloudfunctions` 目录下的每个云函数文件夹
# 2. 选择「上传并部署：云端安装依赖」
# 3. 依次部署以下8个云函数：
   - `addressManager` - 地址管理
   - `cartManager` - 购物车管理
   - `drinkManager` - 饮品管理（含分类管理）
   - `favoriteManager` - 收藏管理
   - `initData` - 初始化数据
   - `orderManager` - 订单管理
   - `shopManager` - 店铺管理
   - `userManager` - 用户管理

## 六、启动项目
# 点击微信开发者工具的普通编译按钮，编译成功后，便可正常访问小程序。

# 体验版小程序
# 用微信开发者工具打开项目后，点击上传按钮，备注是哪个版本的，即可通过，然后在手机上搜索该小程序即可正常访问

# 正式版小程序
# 上传到体验版后，再去到微信公众平台，选择自己申请的小程序进去，然后在首页》版本管理》前往版本管理》开发版本》提交审核。等待审核通过后，即可访问正式版本小程序。
